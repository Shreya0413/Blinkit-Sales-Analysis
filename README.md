# 🚀 Blinkit Sales Analysis – Power BI Dashboard Project

This project presents a comprehensive analysis of Blinkit's sales, customer behavior, marketing campaigns, delivery performance, and inventory insights using interactive Power BI dashboards.

## 📊 Project Overview

The goal of this project is to analyze multi-dimensional sales and operational data from Blinkit to help make data-driven business decisions. It covers data from customers, products, campaigns, delivery, stock, and order transactions.

The dashboard answers 30 curated business questions across basic, intermediate, and advanced levels through interactive visualizations.

## ✅ Key Features

- **Sales Analysis** – Order trends, average order value, customer lifetime value
- **Customer Insights** – Top customers, customer segmentation, retention rate
- **Delivery Performance** – On-time vs delayed deliveries, delivery time analysis
- **Campaign Tracking** – Spend vs revenue, ROAS, funnel conversion
- **Inventory Monitoring** – Stock levels, damaged stock, top-moving products
- **Advanced DAX Measures** – ROAS, gross profit, delivery delays, most ordered product
- **Time Series & Heatmaps** – Daily order trends, geographic order distribution
- **Dynamic Filtering & Slicers** – Delivery status, customer segments

## 🧠 Business Questions Answered

The project answers 30 structured questions including:

- What is the average order value and how does it trend over time?
- Which customers are the most loyal?
- How effective are marketing campaigns in generating revenue?
- What is the Return on Ad Spend (ROAS)?
- Where are most orders geographically concentrated?
- What products are most profitable or most frequently ordered?

## 📝 How to Use

1. Clone this repository.
2. Open the `.pbix` file in Power BI Desktop.
3. Ensure data source paths (if local files are used) are correctly mapped.
4. Explore each page of the dashboard to view answers to the listed questions.
